No entanto, escolher o Linux empresarial certo é crucial para combinar as arquiteturas de hardware, componentes de microchip e plataformas de container ideais para uma implantação eficaz de tecnologias emergentes. O Red Hat Enterprise Linux 8 facilita a adoção de tecnologias emergentes com suporte para inovações de hardware, como as GPUs, o que pode auxiliar as cargas de trabalho de aprendizado de máquina. O suporte para GPU oferece mais opções ao desenvolver e executar aplicações tradicionais ou em containers.

Os servidores DGX-1 e DGX-2 da NVIDIA, criados para fornecer soluções avançadas para os desafios complexos de inteligência artificial, são certificados para o Red Hat Enterprise Linux. Os sistemas DGX têm recursos avançados de aceleração de carga de trabalho baseados em GPU. O Red Hat Enterprise Linux é otimizado para execução nesses sistemas por meio de perfis com ajustes personalizados.

O Red Hat Enterprise Linux 8 oferece suporte a implantação e gerenciamento de containers NGC acelerados com GPU no Red Hat OpenShift. Esses containers de inteligência artificial contam com stacks de software e drivers integrados para executar frameworks de aprendizado de máquina otimizadas para GPU, como TensorFlow, Caffe2, PyTorch, MXNet e outras.

Ferramentas de container que garantem seu poder de escolha

As ferramentas de container do Red Hat Enterprise Linux 8 permitem que você personalize seus sistemas para localizar, executar e compartilhar containers com ferramentas compatíveis com os padrões da Open Container Initiative (OCI). Com mais opções de escolha e suporte ampliado para aplicações em containers, você pode implementar soluções empresariais quando e como quiser.

Junte-se à comunidade de desenvolvedores

  • Podman
  • Buildah
  • Skopeo

Use o Podman para executar containers. Podman é uma ferramenta de linha de comando sem daemon que permite criar e gerenciar imagens de container diretamente. Ao permitir o trabalho com containers sem precisar de um ambiente de execução, você pode controlar melhor as permissões concedidas aos componentes de software.

Use o Buildah para criar containers. Buildah é uma ferramenta de build que oferece controle sobre como as camadas de imagens são compostas e como os dados são acessados durante as compilações. Para usar o Buildah, não é necessário ter um ambiente de execução de containers nem acesso raiz. O Buildah não inclui ferramentas de build na imagem em si, o que reduz o tamanho das imagens criadas.

Use o Skopeo para localizar e compartilhar containers. Skopeo é um utilitário flexível que permite a migração, assinatura e verificação de imagens entre servidores de registro e hosts de containers. O Skopeo pode ser usado para inspecionar imagens fora do seu sistema

Obtenha o resumo técnico para administradores de sistema

  • Fluxo de aplicações
  • Console da web
  • Segurança

Com o fluxo de aplicações, você pode oferecer as aplicações que os desenvolvedores precisam, independentemente das versões do sistema operacional. Use o fluxo de aplicações para separar aplicações do sistema operacional. Assim, você pode atualizar aplicações sem precisar esperar a próxima versão principal do sistema operacional.

O novo console da web fornece uma interface gráfica baseada em navegador para gerenciar o sistema Red Hat Enterprise Linux. Com o console da web, usuários com menos experiência em Linux podem administrar máquinas locais, remotas ou virtuais sem usar a linha de comando. O Red Hat Enterprise Linux ajuda a aprimorar o gerenciamento de identidade. Os componentes do gerenciamento de identidade se integram ao console da web para fornecer a facilidade do logon único (SSO) e o gerenciamento de hosts individuais.

Os controles de segurança integrados, como as políticas de criptografia do sistema com suporte para OpenSSL 1.1.1 e TLS 1.3, ajudam a manter a conformidade da criptografia. O Red Hat Enterprise Linux 8 também minimiza a superfície de ataque ao implantar somente os pacotes necessários para sua carga de trabalho. O Red Hat Enterprise Linux 8 traz melhorias na política de segurança dos controles de acesso obrigatórios do Security-Enhanced Linux (SELinux). Incluído na subscrição, o Red Hat Insights permite a você detectar de maneira proativa problemas e vulnerabilidades do sistema relacionados a segurança, disponibilidade, desempenho e escalabilidade, além de criar um plano de correções.
