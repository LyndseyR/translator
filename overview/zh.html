支持新兴技术

想要引领创新，配置合适的硬件架构、微芯片组件和容器平台，实现对新兴技术的有效部署需要采用合适的企业级 Linux。红帽企业 Linux 8 已获得跨架构和跨环境支持，为您提供一致且稳定的操作系统，可适应机器学习、预测分析、物联网（IoT），边缘计算和大数据工作负载。

  • 机器学习
  • NVIDIA DGX-1 和 DGX-2
  • 适用 AI 的容器

红帽企业 Linux 8 减少了采用新兴技术的障碍，支持 GPU 等硬件创新，可协助机器学习工作负载。在开发和运行传统和容器化应用时，GPU 支持向多选项过渡。

NVIDIA 的 DGX-1 和 DGX-2 服务器针对复杂 AI 挑战提供强大解决方案而设计，已经过红帽企业 Linux 认证。 DGX 系统具备先进的基于GPU 工作负载加速功能。红帽企业 Linux 通过提供定制的调优配置文件进行优化，可在这些系统上运行。

红帽企业 Linux 8 已获得相应支持，可在红帽 OpenShift 上部署和管理 GPU 加速的 NGC 容器。这些 AI 容器提供集成软件堆栈和驱动程序，以运行 GPU 优化的机器学习框架，如 TensorFlow、Caffe2、PyTorch、MXNet 等。

容器工具让您有更多的选择

红帽企业 Linux 8 提供容器工具，支持您定制系统，以便通过其他开放容器计划（OCI）标准兼容工具来查找、运行、构建和共享容器。凭借容器化应用带来的更多选择与支持，您可以随时按需实施业务解决方案。

加入开发人员社区

  • Podman
  • Buildah
  • Skopeo

使用 Podman 来运行容器。 Podman 是一个无守护进程的命令行工具，支持您直接创建和管理容器镜像。通过在不需要运行时环境的情况下使用容器，您可以实现对软件组件授权的更严格控制。

使用 Buildah 来构建容器。 Buildah 是一个构建工具，支持您在构建期间对镜像层的运行方式和数据的访问方式实施控制。使用 Buildah 不需要容器运行时，也不需要根访问。 Buildah 不包括镜像本身的内部构建工具，从而降低您所构建镜像的大小。

使用 Skopeo 来查找和共享容器。 Skopeo 是一个灵活的实用程序，可在注册表服务器和容器主机之间迁移、签名和验证镜像。您可以使用 Skopeo 来检查系统外部镜像

获取系统管理员的技术简介

  • 应用流
  • Web 控制台
  • 安全性

从现在开始，您可以为开发人员提供所需要的应用，同时保持操作系统发布的独立性。 借助应用流能将应用与基本操作系统分开，可以随时更新应用，而不需等待操作系统的下一个主要版本。

新的 Web 控制台可提供一个基于浏览器的图形界面，用于管理您的红帽企业 Linux 系统。有了 Web 控制台，即使是经验不足的 Linux 用户也可管理本地、远程和虚拟机，而无需命令行。 红帽企业 Linux 有助于改善身份管理。身份管理组件与 Web 控制台集成到一起，方便执行单点登录（SSO）和单个主机管理。

内置的安全控件，比如支持 OpenSSL 1.1.1 和 TLS 1.3 的全系统加密策略有助于保障加密的合规性。红帽企业 Linux 8 还可通过仅部署用于支持工作负载所需的软件包来尽可能地缩小攻击面。 此外，红帽企业 Linux 8 还针对安全增强型 Linux®（SELinux）强制访问控件改进了安全策略。 利用红帽智能分析工具（目前包含在订阅内）来主动检测安全性、可用性、性能和可扩展性方面的系统问题和漏洞，并构建补救计划。
